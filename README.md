# 智投 · 股票深度分析平台

## 功能介绍
本平台提供股票深度分析服务，初始包含三只股票的完整分析：
- **嘉美包装 (002969)**：食品包装行业龙头企业，近期控制权变更
- **银禧科技 (300221)**：改性塑料和智能照明领域领先企业  
- **宏微科技 (688711)**：功率半导体器件制造商，IGBT模块市占率全国第一

### 核心功能
- **深度基本面分析**：公司概况、财务数据、行业地位
- **实时新闻整合**：每只股票整合不超过20条相关新闻，带权重打分（22-29分）
- **K线图与成交量**：基于历史数据生成的交互式K线图表
- **AI走势预测**：结合新闻权重和趋势分析，给出当日走势预测
- **动态股票管理**：支持添加/删除股票，新增股票自动进行完整分析

## 本地使用说明
1. 确保已安装Python 3.x
2. 运行 `python server.py` 启动本地服务器
3. 在浏览器中访问 `http://localhost:8000`

## 技术栈
- **前端框架**：HTML5 + Tailwind CSS
- **图标库**：Font Awesome
- **图表库**：ECharts
- **部署方式**：静态网站（无需后端服务器）

## 文件说明
- `index.html`：主网站文件
- `server.py`：本地开发服务器脚本
- `package.py`：网站打包脚本
- `deploy.md`：在线部署配置说明

## 注意事项
- 网站依赖CDN资源（Tailwind CSS、Font Awesome、ECharts），需要网络连接
- 新闻数据和K线数据为模拟数据，仅用于演示功能
- 实际部署时建议替换为真实API数据源
